# Style Guide（語氣、中文解說、德語例句）

> 共用流程見 **`docs/generation-brief.md`**。對應 **`grammatik-roadmap.md`** §5 與全檔精神。

---

## 1. 語言

- 解說使用 **繁體中文**。  
- 德語術語（如 *Nominativ、Perfekt、Konjunktiv*）首次出現時附 **白話說明**。  
- 先給直覺理解，再給較精準文法表述。

### 1.1 台灣繁體中文用語

- 文法術語以 **台灣教材慣用說法** 為準：例如動詞變化時不變的基底稱 **動詞詞幹** 或 **詞幹**（對應德語 *Stamm*／英語 *stem*），**不**用「字幹」。與「**詞尾**」並列時前後用語一致。  
- 其他與陸港用語易混者，優先台灣習慣（如 **軟體、影片、預設、資料夾、程式碼**；避免「軟件、視頻、默認」等大陸詞）。

---

## 2. 例句

- **自然、短、可替換**；生活場景為主。  
- 優先主題：自我介紹、學校／工作、家庭、飲食、時間、交通、購物、日常計畫。  
- 避免例句中出現**大量未教文法**。

---

## 3. 教材語氣

- 清楚、友善、不故作艱深；**先鼓勵、後修正**。  
- 可用：**「先這樣用」**、**「這裡注意格」**、**「先會造句，再細分例外」**。  
- 德語例外多，**避免同一課塞過多例外**。

### 3.1 解說深度與口吻（Core Concept）

- **Core Concept** 目標是讓自學者讀完能回答：**這一課的新規則是什麼、為什麼現在要學、跟上一課差在哪**。避免只有定義句、沒有比喻或場景。  
- 可採：**一句話打氣** → **生活例子** → **規則（含德語術語）** → **「先記這個就好」的邊界**（其餘下課再說）。風格可對齊語言庫內其他語言教材（如日文專案 `JP/units`）的**輕快分段**，但不要為了可愛而犧牲資訊量。  
- **格位與介詞**：說明「為什麼這課要出現某介詞」（例如 **mit** 表「一起／用…工具／跟誰」），並標明該介詞支配 **Dat.** 等，避免無來由硬塞例句。  
- **近義代名詞**（如 **etwas**／**jemand-**）：用 **一兩條決策句** 說明「什麼時候用哪一形」（依 **動詞／介詞要求的格**），必要時附迷你對照表。

### 3.2 德語格名與縮寫（全專案一致）

- 首次於該 unit 出現時：**中文 + 德語 + 縮寫**，例如「**賓格**（**Akkusativ**，**Akk.**）」。  
- **主格**（**Nominativ**，**Nom.**）、**與格／三格**（**Dativ**，**Dat.**）、**屬格**（**Genitiv**，**Gen.**）同理。  
- 表格欄名優先寫 **德語格名（縮寫）**，必要時加極短中文提示。

---

## 4. 錯誤處理（德語常見方向）

教材中應主動點出（依課次適用）：

- **冠詞與格**（der/die/das、ein、變格）混用。  
- **動詞位置**（直陳句第二位、從句尾位等）錯置。  
- **介詞支配格**（auf mit zu 等）與中文「在／到」直譯混淆。  
- **可分離動詞**前綴位置、**框式結構**初學誤判。  
- 英語或中文語序硬套德語。
