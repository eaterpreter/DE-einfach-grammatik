# 被動（一）：現在式與過去式 **werden + PII**

**Stage 8 · Unit 8-1 · Lektion 29** — **Vorgangspassiv** 基本形（**§3.3 L29**）

---

## Learning Goal

- 能寫出**過程被動**（**Vorgangspassiv**）：**Präsens** **werden + Partizip II（PII）**、**Präteritum** **wurde(n) + PII**。  
- 能將主動句改為被動時，把**原受詞**提升為**主語（Nom.）**，動作者可省略或用 **von + Dat.** 帶出。  
- 能辨識框式：**變位之 werden** 在第二位（或問句第一位），**PII** 在**句尾**。

---

## Core Concept

被動強調「**事情被做**」，不必說出誰做。**構成**：**werden**（依新主語變位）＋ **PII**。  
**現在**：**Das Buch wird gelesen.**  
**過去（敘事常見）**：**Das Buch wurde gelesen.**

主動 **Man liest das Buch.**／**Sie liest das Buch.** → 被動 **Das Buch wird gelesen.**（**das Buch** 原為 **Akk.**，變主語後用 **Nom.**）

動作者（可有可無）：**von + Dat.** **Die Musik wurde von einem DJ gespielt.**  
（**durch** 表「經由管道」等，書上另述；**8-5** 再對照 **man／Zustandspassiv**。）

**本課邊界**：**不及物動詞**（無四格受詞者）通常**不能**改成「**werden + PII**」的過程被動；若語意上要淡化施事，可用 **man** 主動句或換句型（細部與 **sein** 狀態被動等見 **8-5**）。

---

## Key Forms / Rules

| 時態 | 形態 |
|------|------|
| **Präsens** | **wird / werden / … + PII**（句尾） |
| **Präteritum** | **wurde / wurden + PII** |

- **PII** 位置：主句內與 **werden** 形成**句框**（與 **Perfekt** 之 **haben + PII** 類比）。  
- 原主語不省略時：**von dem Lehrer, von der Künstlerin**（**Dat.**）。

---

## Example Sentences

### ① **Präsens** 被動

**Das Buch wird gelesen.** （那本書正在被讀／有人讀那本書。）  
**Die Tür wird geöffnet.** （門正在被打開。）  
**Viele E-Mails werden beantwortet.** （很多電子郵件正在被回覆。）  
**Der Kuchen wird gebacken.** （蛋糕正在烤。）  
**Die Aufgabe wird heute erledigt.** （這項任務今天會被完成。）

### ② **Präteritum** 被動

**Das Haus wurde verkauft.** （那棟房子被賣掉了。）  
**Die Nachricht wurde veröffentlicht.** （那則消息被發表了。）  
**Die Fenster wurden geschlossen.** （窗戶被關上了。）  
**Das Problem wurde besprochen.** （那個問題被討論過了。）  
**Die Gäste wurden begrüßt.** （客人受到了歡迎。）

### ③ **von + Dat.**（動作者）

**Das Lied wurde von einem Chor gesungen.** （那首歌由合唱團演唱。）  
**Der Film wird von vielen Leuten geliebt.** （那部片受到許多人喜愛。）  
**Die Entscheidung wurde von der Regierung getroffen.** （那項決定由政府做出。）  
**Der Text wird von der Lehrerin korrigiert.** （那篇文字由老師批改。）  
**Das Essen wurde von den Eltern vorbereitet.** （那餐由父母準備。）

### ④ 與主動對照（短句）

**Man baut ein neues Museum.** → **Ein neues Museum wird gebaut.** （大家在蓋一座新博物館。→ 一座新博物館正在蓋。）  
**Sie schreiben den Bericht.** → **Der Bericht wird geschrieben.** （他們在寫報告。→ 報告正在被寫。）  
**Er hat die Tür repariert.**（主動完成式）→ **Die Tür wurde repariert.**（被動過去敘事）（他修好了門。→ 門被修好了。）  
**Wir diskutieren das Thema.** → **Das Thema wird diskutiert.** （我們在討論那個主題。→ 那個主題正在被討論。）  
**Ihr habt die Regeln erklärt.** → **Die Regeln wurden erklärt.** （你們解釋了規則。→ 規則被說明過了。）

## Sentence Breakdown

### ① **Präsens** 被動

1. **Das Buch wird gelesen.** — **Das Buch**（**Nom.**，原受詞升格）｜**wird**（**V2**）｜**gelesen**（**PII** 句尾）。
2. **Die Tür wird geöffnet.** — **wird**＋**geöffnet**；單數主語 **Die Tür** 配 **wird**。
3. **Viele E-Mails werden beantwortet.** — 複數主語配 **werden**；**beantwortet** 句尾。
4. **Der Kuchen wird gebacken.** — **Der Kuchen**（**Nom.**）｜**wird gebacken** 框式。
5. **Die Aufgabe wird heute erledigt.** — **heute** 時間狀語常在 **V2** 前或後；核心仍 **wird … erledigt**。

### ② **Präteritum** 被動

6. **Das Haus wurde verkauft.** — **wurde**（過去敘事）＋**verkauft**（**PII**）。
7. **Die Nachricht wurde veröffentlicht.** — 單數 **wurde**＋**veröffentlicht**。
8. **Die Fenster wurden geschlossen.** — 複數 **wurden**＋**geschlossen**。
9. **Das Problem wurde besprochen.** — **wurde besprochen** 框式。
10. **Die Gäste wurden begrüßt.** — **wurden** 與複數主語一致。

### ③ **von + Dat.**（動作者）

11. **Das Lied wurde von einem Chor gesungen.** — **von einem Chor**（**von + Dat.**）表行為者；**wurde … gesungen**。
12. **Der Film wird von vielen Leuten geliebt.** — **von vielen Leuten**（**Dat.**）；現在 **wird … geliebt**。
13. **Die Entscheidung wurde von der Regierung getroffen.** — **von der Regierung**；**wurde getroffen**（可分 **treffen** → **getroffen**）。
14. **Der Text wird von der Lehrerin korrigiert.** — **von der Lehrerin**；**wird korrigiert**。
15. **Das Essen wurde von den Eltern vorbereitet.** — **von den Eltern**；**wurde vorbereitet**。

### ④ 與主動對照（短句）

16. **Man baut … → Ein neues Museum wird gebaut.** — 主動省略主語常用 **man**；被動以原 **Akk.** **ein neues Museum** 為新 **Nom.**；**wird gebaut**。
17. **Sie schreiben … → Der Bericht wird geschrieben.** — **Der Bericht** 升格為主語；**wird geschrieben**。
18. **Er hat … → Die Tür wurde repariert.** — 主動 **Perfekt** 對應敘事被動常用 **Prät.** **wurde repariert**（非 **hat … repariert** 的直譯被動）。
19. **Wir diskutieren … → Das Thema wird diskutiert.** — **Das Thema**（**Nom.**）｜**wird diskutiert**。
20. **Ihr habt … → Die Regeln wurden erklärt.** — 複數 **Die Regeln** 配 **wurden erklärt**。

## Common Mistakes

1. **錯** **Das Buch ist gelesen.**（當「正在被讀」）→ **正** **Das Buch wird gelesen.** — 過程被動用 **werden**，不用 **sein**（**sein + PII** 多為狀態被動，見 **8-5**）。  
2. **錯** **Die Tür wird öffnen.** → **正** **Die Tür wird geöffnet.** — 被動須用 **PII**，不是原形。  
3. **錯** **Viele E-Mails wird beantwortet.** → **正** **Viele E-Mails werden beantwortet.** — 複數主語用 **werden**，非 **wird**。  
4. **錯** **Der Kuchen werden gebacken.** → **正** **Der Kuchen wird gebacken.** — 單數主語配 **wird**。  
5. **錯** **Die Aufgabe wird heute erledigen.** → **正** **Die Aufgabe wird heute erledigt.** — 句尾須 **PII**。  
6. **錯** **Das Haus wird verkauft.**（要表「當時賣掉」）→ **正** **Das Haus wurde verkauft.** — 敘事過去用 **wurde**，非現在 **wird**。  
7. **錯** **Die Nachricht wurden veröffentlicht.** → **正** **Die Nachricht wurde veröffentlicht.** — 單數主語配 **wurde**。  
8. **錯** **Die Fenster wurde geschlossen.** → **正** **Die Fenster wurden geschlossen.** — 複數主語配 **wurden**。  
9. **錯** **Das Problem wurde besprochen werden.** → **正** **Das Problem wurde besprochen.** — **Prät.** 被動只需 **wurde + PII**，勿再加 **werden**。  
10. **錯** **Die Gäste wurde begrüßt.** → **正** **Die Gäste wurden begrüßt.** — 主語複數，助動詞用 **wurden**。  
11. **錯** **Das Lied wurde von einem Chor singen.** → **正** **Das Lied wurde von einem Chor gesungen.** — 被動謂語用 **PII**。  
12. **錯** **Der Film wird von vielen Leuten lieben.** → **正** **Der Film wird von vielen Leuten geliebt.** — **von + Dat.** 後仍接 **PII** 結構。  
13. **錯** **Die Entscheidung wurde von die Regierung getroffen.** → **正** **Die Entscheidung wurde von der Regierung getroffen.** — **von** 支配 **Dat.** **der Regierung**。  
14. **錯** **Der Text wird von der Lehrerin korrigierte.** → **正** **Der Text wird von der Lehrerin korrigiert.** — 主要動詞用 **PII**，勿誤用過去式詞尾。  
15. **錯** **Das Essen wurde von den Eltern vorbereiten.** → **正** **Das Essen wurde von den Eltern vorbereitet.** — **PII** **vorbereitet**。  
16. **錯** **Ein neues Museum ist gebaut.**（表「正在蓋」）→ **正** **Ein neues Museum wird gebaut.** — 表建造過程用 **wird + PII**。  
17. **錯** **Der Bericht wird schreiben.** → **正** **Der Bericht wird geschrieben.** — **PII** **geschrieben**。  
18. **錯** **Die Tür wurde reparieren.** → **正** **Die Tür wurde repariert.** — **PII** **repariert**。  
19. **錯** **Das Thema wird diskutieren.** → **正** **Das Thema wird diskutiert.** — **diskutiert** 為 **PII**。  
20. **錯** **Die Regeln wurde erklärt.** → **正** **Die Regeln wurden erklärt.** — 複數主語 **wurden**。

## Quick Review

- 過程被動：**werden + PII**；現在 **wird／werden**，過去 **wurde／wurden**。  
- 原 **Akk.** 受詞 → 新 **Nom.** 主語；動作者 **von + Dat.**。  
- **8-2**：**Perfekt** 被動等複合時態。
