# 不定冠詞與物主冠詞：形容詞變格（混合變化）

**Stage 6 · Unit 6-3 · Lektion 21** — **ein／kein／mein／dein…** 後的形容詞（**§3.3 L21**）

---

## Learning Goal

- 能辨識 **ein, kein** 與**物主冠詞**（**mein, dein, sein…**）後的**混合變化**（**gemischte Deklination**）。  
- 能對照 **6-1**（弱）與 **6-2**（強）：混合變化在「**無詞尾冠詞**」格位上較像強變化，其餘多像弱變化。  
- 能正確寫出 **ein alter Freund／einen alten Freund／mein kleines Kind** 等片語。

---

## Core Concept

**Ein** 與 **kein** 在部分格沒有詞尾（如 **ein** 陽性 **Nom.**、中性 **Nom./Akk.**）。**誰來補格標？**— 緊鄰的形容詞要帶**較明顯**的結尾（類似強變化）。若冠詞已有清楚標記（如 **eine, einen, einem**），形容詞則多用 **-e／-en**，像弱變化。

**物主冠詞**（**mein, dein, sein, ihr, unser, euer…**）行為與 **ein／kein** 同組。  
本課與 **6-1／6-2** 合起來，即 roadmap 所謂**三狀況比較**的第三塊。

---

## Key Forms / Rules

**混合變化（精簡記法）**

- **陽性**：**ein großer Mann**（**Nom.**）／**einen großen Mann**（**Akk.**）／**einem großen Mann**（**Dat.**）  
- **陰性**：**eine kleine Katze** — **Akk./Nom.** 形容詞常 **-e**；**Dat.** **kleinen**  
- **中性**：**ein kleines Kind**（**Nom.**）／**ein kleines Kind**（**Akk.** 同形）／**einem kleinen Kind**  
- **複數**：**keine roten Äpfel**；**Dat.** **keinen roten Äpfeln**

---

## Example Sentences

### ① **ein／kein**（陽性／中性）

**Ein alter Freund ruft an.** （一位老朋友打電話來。）
**Ich sehe einen alten Freund.** （我看到一位老朋友。）
**Mit einem alten Freund gehe ich essen.** （我要和一位老朋友出去吃飯。）
**Kein großes Problem!** （問題不大！）
**Wir haben kein großes Auto.** （我們沒有大車。）

### ② **ein／kein**（陰性）

**Eine junge Frau wartet.** （一位年輕女子正在等待。）
**Ich kenne eine junge Frau.** （我認識一個年輕女子。）
**Ich helfe einer jungen Frau.** （我正在幫助一位年輕女士。）
**Keine schlechte Idee!** （不錯的主意！）
**Das ist keine schlechte Idee.** （這不是一個壞主意。）

### ③ **物主冠詞**

**Mein kleiner Bruder lacht.** （我的弟弟笑了。）
**Ich sehe deinen kleinen Bruder.** （我看到你的弟弟了。）
**Das ist ihr neues Fahrrad.** （這是她的新自行車。）
**Wir danken unserem netten Nachbarn.** （我們感謝我們的好鄰居。）
**Euer großer Hund bellt.** （你的大狗在吠叫。）

### ④ 複數與對照

**Keine roten Rosen hier?** （這裡沒有紅玫瑰嗎？）
**Ich kaufe keine roten Rosen.** （我不買紅玫瑰。）
**Mit meinen alten Freunden trinke ich Kaffee.** （我和我的老朋友一起喝咖啡。）
**Ein guter Tag und eine gute Nacht.** （美好的一天和美好的夜晚。）
**Sein neues Handy liegt auf dem Tisch.** （他的新手機放在桌上。）

---

## Sentence Breakdown

### ① **ein／kein**（陽性／中性）

1. **Ein alter Freund ruft an.** — **ein**（無尾）+ **alter**（混合：陽 **Nom.** **-er**）+ **Freund**；**ruft an**。  
2. **Ich sehe einen alten Freund.** — **einen** + **alten**（**Akk.** **-en**）。  
3. **Mit einem alten Freund gehe ich essen.** — **einem** + **alten**（**Dat.** **-en**）。  
4. **Kein großes Problem!** — **kein** + **großes**（中 **Nom.** **-es**）。  
5. **Wir haben kein großes Auto.** — **kein großes**（中 **Akk.** 同 **Nom.** 形）。

### ② **ein／kein**（陰性）

1. **Eine junge Frau wartet.** — **eine** + **junge**（**-e**）。  
2. **Ich kenne eine junge Frau.** — **eine junge**（**Akk.** 同形）。  
3. **Ich helfe einer jungen Frau.** — **einer** + **jungen**（**Dat.** **-en**）。  
4. **Keine schlechte Idee!** — **keine** + **schlechte**（陰 **-e**）。  
5. **Das ist keine schlechte Idee.** — **keine schlechte** + **Idee**。

### ③ **物主冠詞**

1. **Mein kleiner Bruder lacht.** — **mein**（無尾）+ **kleiner**（陽 **Nom.** **-er**）。  
2. **Ich sehe deinen kleinen Bruder.** — **deinen** + **kleinen**（**Akk.** **-en**）。  
3. **Das ist ihr neues Fahrrad.** — **ihr** + **neues**（中 **Nom.** **-es**）。  
4. **Wir danken unserem netten Nachbarn.** — **unserem** + **netten** + **Nachbarn**（**Dat.**）。  
5. **Euer großer Hund bellt.** — **euer** + **großer**（陽 **Nom.**）。

### ④ 複數與對照

1. **Keine roten Rosen hier?** — **keine** + **roten**（複數 **-en**）。  
2. **Ich kaufe keine roten Rosen.** — **keine roten**（**Akk.** 複數）。  
3. **Mit meinen alten Freunden trinke ich Kaffee.** — **meinen** + **alten** + **Freunden**（**Dat.** 複數）。  
4. **Ein guter Tag und eine gute Nacht.** — 陽 **guter** + **Tag**；陰 **eine gute** + **Nacht**。  
5. **Sein neues Handy liegt auf dem Tisch.** — **sein** + **neues**；**auf dem Tisch**（**Dat.**）。

## Common Mistakes

### ① **ein／kein**（陽性／中性）

1. **錯** Ein alte Freund ruft an. → **正** Ein alter Freund ruft an. — **ein** 無詞尾時形容詞帶 **-er**（陽 **Nom.**）。  
2. **錯** Ich sehe ein alten Freund. → **正** … einen alten Freund. — **Akk.** **einen** + **-en**。  
3. **錯** Mit ein alten Freund … → **正** Mit einem alten Freund … — **Dat.** **einem** + **-en**。  
4. **錯** Kein große Problem! → **正** Kein großes Problem! — 中性 **Nom.** **-es**。  
5. **錯** Wir haben kein großen Auto. → **正** … kein großes Auto. — 中性 **Akk.** **-es**。

### ② **ein／kein**（陰性）

1. **錯** Eine jungen Frau wartet. → **正** Eine junge Frau wartet. — **eine** 後形容詞 **-e**。  
2. **錯** Ich kenne eine jungen Frau. → **正** … eine junge Frau. — **Akk.** 同 **Nom.** 形。  
3. **錯** Ich helfe eine junge Frau. → **正** … einer jungen Frau. — **Dat.** **einer** + **-en**。  
4. **錯** Kein schlechte Idee! → **正** Keine schlechte Idee! — 陰性用 **keine**。  
5. **錯** Das ist kein schlechte Idee. → **正** … keine schlechte Idee. — **keine**（陰）。

### ③ **物主冠詞**

1. **錯** Mein kleine Bruder lacht. → **正** Mein kleiner Bruder lacht. — **mein** 後陽 **Nom.** **-er**。  
2. **錯** Ich sehe dein kleinen Bruder. → **正** … deinen kleinen Bruder. — **deinen** + **-en**。  
3. **錯** Das ist ihr neu Fahrrad. → **正** … ihr neues Fahrrad. — 中性 **neues**。  
4. **錯** Wir danken unser netten Nachbarn. → **正** … unserem netten Nachbarn. — **Dat.** **unserem**。  
5. **錯** Euer große Hund bellt. → **正** Euer großer Hund bellt. — 陽 **Nom.** **großer**。

### ④ 複數與對照

1. **錯** Kein rote Rosen hier? → **正** Keine roten Rosen hier? — 複數 **keine** + **-en**。  
2. **錯** Ich kaufe kein rote Rosen. → **正** … keine roten Rosen. — **keine roten**。  
3. **錯** Mit meine alten Freunden … → **正** Mit meinen alten Freunden … — **Dat.** **meinen**。  
4. **錯** Ein gute Tag und eine gut Nacht. → **正** Ein guter Tag und eine gute Nacht. — 陽／陰詞尾有別。  
5. **錯** Sein neu Handy liegt … → **正** Sein neues Handy liegt … — 中性 **neues**。

## Quick Review

- **ein／kein／物主**後多為**混合變化**：看冠詞有無「露出」格尾，決定形容詞負擔。  
- **Nom.** 陽 **-er**、中 **-es**；**Akk.** 陽常 **-en**。  
- **L21** 三課：**6-1 定冠**、**6-2 無冠**、**6-3 不定／物主**。
