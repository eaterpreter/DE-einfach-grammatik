# 被動（四）：從句與情態動詞

**Stage 8 · Unit 8-4 · Lektion 29** — **dass／weil** 等末位；**können／müssen … + Infinitiv** 被動（**§3.3 L29**）

---

## Learning Goal

- 能在**從句**把變位動詞放在**句尾**：**…, dass das Buch gelesen wird.**  
- 能組合**情態動詞＋被動不定式**：**Das Buch kann gelesen werden.**  
- 能辨識從句內**兩個原形**置末：**…, dass es gemacht werden muss.**

---

## Core Concept

**從句**（**dass, weil, ob** 等）中，**變位動詞**在**從句最後**。被動亦然：

**Ich hoffe, dass die Arbeit bald erledigt wird.**  
**Er fragt, ob das Fenster geöffnet werden kann.**

**情態動詞＋被動**：情態動詞變位占主句 **V2**，被動用**原形片語** **PII + werden** 置於**句尾**：

**Das Buch kann morgen gelesen werden.**  
**Die Tür musste geschlossen werden.**

若情態動詞本身在**從句**內變位，則順序為：**…, dass das Buch gelesen werden kann.**（**PII** 在前，**werden** 與情態原形在後—**verb cluster** 規則依書細化，初階記「**被動相關原形在末段**」。）

---

## Key Forms / Rules

- 主句：**Das Problem muss gelöst werden.**  
- 從句：**…, weil das Problem gelöst werden muss.**  
- **Perfekt** 被動＋情態（進階，先認識）：**Es hat gemacht werden müssen.**（書上若出現，對照 **8-2**。）

---

## Example Sentences

### ① **dass** 從句（被動）

**Ich weiß, dass der Text korrigiert wird.** （我知道那篇文字正在被批改。）  
**Sie sagt, dass die Tickets verkauft wurden.** （她說票已經賣掉了。）  
**Wir hoffen, dass die Nachricht verstanden wird.** （我們希望那則消息能被理解。）  
**Er glaubt, dass das Haus schon gebaut worden ist.** （他相信那棟房子已經蓋好了。）  
**Ihr wisst, dass die Regeln erklärt werden müssen.** （你們知道規則必須被說明。）

### ② **weil／ob** 從句

**Er kommt, weil er eingeladen wurde.** （他來，因為他受到邀請。）  
**Sie fragt, ob das Meeting verschoben wird.** （她問會議是否會延期。）  
**Wir bleiben, weil die Tür geschlossen wurde.** （我們留下，因為門關上了。）  
**Ich weiß nicht, ob die Arbeit schon erledigt worden ist.** （我不知道那項工作是否已經完成。）  
**Sie fragt sich, ob sie informiert werden kann.** （她在想自己能不能被告知。）

### ③ 主句：**情態 + 被動原形**

**Das Buch kann hier ausgeliehen werden.** （這本書可以在這裡借出。）  
**Die Datei muss heruntergeladen werden.** （那個檔案必須下載。）  
**Der Antrag sollte schnell bearbeitet werden.** （那份申請應該盡快處理。）  
**Die Tür darf nicht geöffnet werden.** （門不可以打開。）  
**Das Problem konnte nicht gelöst werden.** （那個問題當時無法解決。）

### ④ 從句內情態＋被動

**…, dass das Buch hier ausgeliehen werden kann.** （……這本書可以在這裡借出。）  
**…, weil die Datei heruntergeladen werden muss.** （……因為那個檔案必須下載。）  
**…, ob der Antrag bearbeitet werden soll.** （……那份申請是否該處理。）  
**Er erklärt, dass die Tür nicht geöffnet werden darf.** （他說明門不可以打開。）  
**Wir wussten, dass das Problem nicht gelöst werden konnte.** （我們當時知道那個問題無法解決。）

## Sentence Breakdown

### ① **dass** 從句（被動）

1. **Ich weiß, dass der Text korrigiert wird.** — 主句 **weiß**（**V2**）；**dass** 從句末 **korrigiert wird**（被動 **PII + 變位 werden**）。
2. **Sie sagt, dass die Tickets verkauft wurden.** — 從句末 **verkauft wurden**（**Prät.** 被動）。
3. **Wir hoffen, dass die Nachricht verstanden wird.** — **verstanden wird** 在從句尾。
4. **Er glaubt, dass das Haus schon gebaut worden ist.** — **Perfekt** 被動：**… gebaut worden ist**（**worden** 在 **ist** 前）。
5. **Ihr wisst, dass die Regeln erklärt werden müssen.** — 從句末 **erklärt werden müssen**（情態＋被動原形塊）。

### ② **weil／ob** 從句

6. **Er kommt, weil er eingeladen wurde.** — **weil** 從句末 **eingeladen wurde**。
7. **Sie fragt, ob das Meeting verschoben wird.** — **ob** 從句末 **verschoben wird**。
8. **Wir bleiben, weil die Tür geschlossen wurde.** — **geschlossen wurde** 句尾。
9. **Ich weiß nicht, ob die Arbeit schon erledigt worden ist.** — **erledigt worden ist**（完成式被動疑問從句）。
10. **Sie fragt sich, ob sie informiert werden kann.** — 末段 **informiert werden kann**（情態＋被動）。

### ③ 主句：**情態 + 被動原形**

11. **Das Buch kann hier ausgeliehen werden.** — **V2** **kann**；句尾 **ausgeliehen werden**（**PII + werden** 原形）。
12. **Die Datei muss heruntergeladen werden.** — **muss**＋**heruntergeladen werden**。
13. **Der Antrag sollte schnell bearbeitet werden.** — **sollte**＋**bearbeitet werden**。
14. **Die Tür darf nicht geöffnet werden.** — **darf nicht**＋**geöffnet werden**。
15. **Das Problem konnte nicht gelöst werden.** — **konnte**＋**gelöst werden**（過去）。

### ④ 從句內情態＋被動

16. **…, dass das Buch hier ausgeliehen werden kann.** — 從句末 **ausgeliehen werden kann**（**PII**—**werden**—情態）。
17. **…, weil die Datei heruntergeladen werden muss.** — **heruntergeladen werden muss**。
18. **…, ob der Antrag bearbeitet werden soll.** — **bearbeitet werden soll**。
19. **Er erklärt, dass die Tür nicht geöffnet werden darf.** — **nicht geöffnet werden darf** 在從句尾。
20. **Wir wussten, dass das Problem nicht gelöst werden konnte.** — **gelöst werden konnte**（過去情態）。

---

## Common Mistakes

1. **錯** **Ich weiß, dass wird der Text korrigiert.** → **正** **Ich weiß, dass der Text korrigiert wird.** — 從句變位動詞在句尾。  
2. **錯** **Sie sagt, dass die Tickets wurden verkauft.** → **正** **Sie sagt, dass die Tickets verkauft wurden.** — 被動 **PII + 助動詞** 順序，**wurden** 在末。  
3. **錯** **Wir hoffen, dass die Nachricht wird verstanden.** → **正** **Wir hoffen, dass die Nachricht verstanden wird.**  
4. **錯** **Er glaubt, dass das Haus ist schon gebaut worden.** → **正** **Er glaubt, dass das Haus schon gebaut worden ist.** — **Perfekt** 被動詞序 **… worden ist**。  
5. **錯** **Ihr wisst, dass die Regeln müssen erklärt werden.** → **正** **Ihr wisst, dass die Regeln erklärt werden müssen.** — 情態在從句尾。  
6. **錯** **Er kommt, weil er wurde eingeladen.** → **正** **Er kommt, weil er eingeladen wurde.** — **PII + wurde**。  
7. **錯** **Sie fragt, ob wird das Meeting verschoben.** → **正** **Sie fragt, ob das Meeting verschoben wird.**  
8. **錯** **Wir bleiben, weil die Tür wurde geschlossen.** → **正** **Wir bleiben, weil die Tür geschlossen wurde.**  
9. **錯** **Ich weiß nicht, ob die Arbeit ist schon erledigt worden.** → **正** **Ich weiß nicht, ob die Arbeit schon erledigt worden ist.**  
10. **錯** **Sie fragt sich, ob sie kann informiert werden.** → **正** **Sie fragt sich, ob sie informiert werden kann.** — 情態句尾。  
11. **錯** **Das Buch kann hier wird ausgeliehen.** → **正** **Das Buch kann hier ausgeliehen werden.** — 主句情態後 **PII + werden**。  
12. **錯** **Die Datei muss herunterladen werden.** → **正** **Die Datei muss heruntergeladen werden.** — **PII** **heruntergeladen**。  
13. **錯** **Der Antrag sollte schnell bearbeiten werden.** → **正** **Der Antrag sollte schnell bearbeitet werden.** — **bearbeitet**。  
14. **錯** **Die Tür darf nicht öffnen werden.** → **正** **Die Tür darf nicht geöffnet werden.** — **geöffnet**。  
15. **錯** **Das Problem konnte nicht lösen werden.** → **正** **Das Problem konnte nicht gelöst werden.** — **gelöst**。  
16. **錯** **…, dass das Buch kann hier ausgeliehen werden.** → **正** **…, dass das Buch hier ausgeliehen werden kann.** — 從句末為 **werden kann**。  
17. **錯** **…, weil die Datei muss heruntergeladen werden.** → **正** **…, weil die Datei heruntergeladen werden muss.**  
18. **錯** **…, ob der Antrag soll bearbeitet werden.** → **正** **…, ob der Antrag bearbeitet werden soll.**  
19. **錯** **Er erklärt, dass die Tür nicht darf geöffnet werden.** → **正** **Er erklärt, dass die Tür nicht geöffnet werden darf.**  
20. **錯** **Wir wussten, dass das Problem konnte nicht gelöst werden.** → **正** **Wir wussten, dass das Problem nicht gelöst werden konnte.**

## Practice

### 文法與詞彙選擇（四選一）

**Q1.** **Ich weiß, dass der Text ___.**  
A. korrigiert wird　B. wird korrigiert　C. korrigiert　D. ist korrigiert  
||**正解：A** korrigiert wird||

**Q2.** **Das Buch kann morgen ___.**  
A. gelesen werden　B. werden gelesen　C. wird gelesen　D. gelesen wird  
||**正解：A** gelesen werden||

**Q3.** **Die Tür musste ___.**  
A. geschlossen werden　B. werden geschlossen　C. wurde geschlossen　D. geschlossen wurde  
||**正解：A** geschlossen werden||

**Q4.** **Sie fragt, ob das Meeting ___.**  
A. verschoben wird　B. wird verschoben　C. verschiebt　D. ist verschoben  
||**正解：A** verschoben wird||

**Q5.** **Er kommt, weil er ___.**  
A. eingeladen wurde　B. wurde eingeladen　C. einlädt　D. eingeladen ist  
||**正解：A** eingeladen wurde||

**Q6.** **Die Datei ___ heruntergeladen werden.**  
A. muss　B. muss werden　C. wird　D. musste werden  
||**正解：A** muss（**Die Datei muss heruntergeladen werden.**）||

**Q7.** **Wir hoffen, dass die Arbeit bald ___.**  
A. erledigt wird　B. wird erledigt　C. erledigt　D. ist erledigt  
||**正解：A** erledigt wird||

**Q8.** **Der Antrag sollte schnell ___.**  
A. bearbeitet werden　B. werden bearbeitet　C. bearbeitet　D. wurde bearbeitet  
||**正解：A** bearbeitet werden||

**Q9.** **Ich weiß nicht, ob sie informiert ___.**  
A. werden kann　B. kann werden　C. wird können　D. kann informiert  
||**正解：A** werden kann||

**Q10.** **Die Tür ___ nicht geöffnet werden.**  
A. darf　B. darf nicht　C. muss nicht　D. kann nicht  
||**正解：A** darf（**Die Tür darf nicht geöffnet werden.**）||

**Q11.** **Er glaubt, dass das Haus schon ___ ist.**  
A. gebaut worden　B. worden gebaut　C. gebaut wurde　D. baut worden  
||**正解：A** gebaut worden||

**Q12.** **…, weil die Regeln ___ müssen.**  
A. erklärt werden　B. werden erklärt　C. erklärt wird　D. erklären werden  
||**正解：A** erklärt werden||

### 句構與語序（完整句四選一）

**Q13.** 哪一句正確？  
A. Ich weiß, dass der Text korrigiert wird.　B. Ich weiß, dass wird der Text korrigiert.　C. Ich weiß, der Text dass korrigiert wird.　D. Ich weiß, dass korrigiert der Text wird.  
||**正解：A**||

**Q14.** 哪一句正確？  
A. Das Buch kann morgen gelesen werden.　B. Das Buch kann wird morgen gelesen.　C. Das Buch morgen kann gelesen werden.　D. Kann das Buch morgen gelesen werden.  
||**正解：A**||

**Q15.** 哪一句正確？  
A. Sie fragt, ob das Fenster geöffnet werden kann.　B. Sie fragt, ob kann das Fenster geöffnet werden.　C. Sie fragt, ob geöffnet werden kann das Fenster.　D. Sie fragt, ob das Fenster kann geöffnet werden.  
||**正解：A**||

**Q16.** 哪一句正確？  
A. Die Datei muss heruntergeladen werden.　B. Die Datei muss werden heruntergeladen.　C. Die Datei heruntergeladen muss werden.　D. Muss die Datei heruntergeladen werden.  
||**正解：A**||

## Quick Review

- 從句：**…, dass/weil … PII + 變位 werden／情態。**  
- 主句情態：**… kann/muss … + PII + werden.**  
- **8-5**：狀態被動、**man**、**von**。
