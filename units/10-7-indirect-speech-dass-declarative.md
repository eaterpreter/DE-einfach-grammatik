# 間接引語：連接詞直述（*dass*）

**Stage 10 · Unit 10-7 · Lektion 39** — **dass**-Satz、動詞位置、時態退一步（**§3.3 L39** 第二段）

---

## Learning Goal

- 能將直述句改為 **Er sagt, dass …** 框架，**從句動詞句末**。  
- 能初步運用**時態退一步**（**Präsens → Präteritum** 等，依主句與書本表）。  
- 能與 **10-6** 直接引語對照改寫。

---

## Core Concept

**間接引語**用 **dass**（或 **ob** 是否句）包住內容，**不再用引號**（同一標準下）：**Er sagt, **dass** er **müde** **ist**.* 若主句**過去**，從句常**退一步**（**… sagte, dass er müde **war**.*）。

教學上先固定：**dass** + **尾位動詞**；人稱代詞依**實際指涉**調整（**Ich → er**）。

---

## Key Forms / Rules

| 直接 | 間接（框架） |
|------|----------------|
| „Ich komme.“ | **Er sagte, dass er käme/komme.**（**KI**／直陳替代見 **10-8**） |
| „Sie ist da.“ | **Er sagte, dass sie da sei/wäre.** |

※ 本課以 **dass + 尾位** 與「退一步」概念為主；**KI** 細節 **10-8**。

---

## Example Sentences

### ① 現在錨點

**Sie sagt, dass sie müde ist.** （她說她累。）  
**Er meint, dass der Plan gut ist.** （他覺得計畫不錯。）  
**Wir glauben, dass es klappt.** （我們相信行得通。）  
**Ich höre, dass du umziehst.** （我聽說你要搬家。）  
**Sie erzählt, dass sie Urlaub macht.** （她說她在放假。）

### ② 主句過去：退一步

**Er sagte, dass er keine Zeit hatte.** （他說他沒時間。）  
**Sie meinte, dass alles in Ordnung war.** （她說一切都好。）  
**Wir hörten, dass der Zug Verspätung hatte.** （我們聽說火車誤點。）  
**Ich erfuhr, dass sie krank war.** （我得知她病了。）  
**Sie antwortete, dass sie später kommen würde.** （她答會晚點來。）

### ③ **dass** 可省略？（先記「不省略」）

**Ich denke, dass das stimmt.** （我覺得那是真的。）  
**Er findet, dass es zu teuer ist.** （他覺得那太貴了。）  
**Sie sagt, dass sie bleibt.** （她說她會留下。）  
**Wir hoffen, dass es aufhört.** （我們希望它會停下來。）  
**Sie befürchtet, dass es regnet.** （她擔心會下雨。）

### ④ 人稱轉換

**Direkt: „Ich helfe dir.“ → Indirekt: Er sagte, dass er mir helfen würde/könnte.** （直接引語改成間接引語時，人稱要跟著指涉改。）  
**„Wir sind da.“ → Sie sagten, dass sie da seien/wären.** （「我們到了」轉成「他們說他們到了」。）  
**„Du bist pünktlich.“ → Ich sagte, dass er pünktlich war.** （把對話中的 **du** 轉成敘述裡的 **er**。）  
**„Ich bin fertig.“ → Sie sagte, dass sie fertig war.** （「我好了」轉成「她說她好了」。）  
**„Es regnet.“ → Er meinte, dass es regnete.** （「在下雨」轉述成過去敘述框架。）

## Sentence Breakdown

### ① 現在錨點（對應 Example ①）

| # | 句 | 重點 |
|---|-----|------|
| 1 | **Sie sagt, dass sie müde ist.** | **dass** + **ist** 尾位；主從人稱一致 **sie** |
| 2 | **Er meint, dass der Plan gut ist.** | **der Plan**（**Nom.**）；從句 **V-letzt** |
| 3 | **Wir glauben, dass es klappt.** | **es** 虛主語；**klappt** 尾位 |
| 4 | **Ich höre, dass du umziehst.** | **du** 指對方；**umziehst** |
| 5 | **Sie erzählt, dass sie Urlaub macht.** | **Urlaub machen** 固定；**macht** 尾位 |

| # | 句 | 重點 |
|---|-----|------|

| 7 | **Sie meinte, dass alles in Ordnung war.** | **alles**；**war** 尾位 |
| 8 | **Wir hörten, dass der Zug Verspätung hatte.** | **Verspätung haben**；**hatte** |
| 9 | **Ich erfuhr, dass sie krank war.** | **erfuhr**（**Prät.**）；**sie** 第三人稱 |
| 10 | **Sie antwortete, dass sie später kommen würde.** | **würde kommen** 尾位框式 |

| # | 句 | 重點 |

| 15 | **Sie befürchtet, dass es regnet.** | **befürchten**；**regnet** |

|---|-----|------|

## Common Mistakes

1. **Sie sagt, sie müde ist.** **錯** → **正** **Sie sagt, dass sie müde ist.** → 初學固定加 **dass**，且從句動詞句末。  
2. **Er meint, der Plan ist gut.** **錯** → **正** **Er meint, dass der Plan gut ist.** → 間接直述須 **dass**-語序，**ist** 在句末。  
3. **Wir glauben, es klappt.** **錯** → **正** **Wir glauben, dass es klappt.** → 同上（本課先不練省略）。  
4. **Ich höre, dass du umziehst du.** **錯** → **正** **Ich höre, dass du umziehst.** → 從句只有一個動詞第二位概念，尾位只有 **umziehst**。  
5. **Sie erzählt, dass sie macht Urlaub.** **錯** → **正** **Sie erzählt, dass sie Urlaub macht.** → **Urlaub machen** 整框置末。  
6. **Er sagte, dass er keine Zeit hat.** **錯** → **正** **Er sagte, dass er keine Zeit hatte.** → 主句過去時，從句常退一步用 **hatte**。  
7. **Sie meinte, dass alles in Ordnung ist.** **錯** → **正** **Sie meinte, dass alles in Ordnung war.** → 錨點一致用 **war**。  
8. **Wir hörten, dass der Zug hat Verspätung.** **錯** → **正** **Wir hörten, dass der Zug Verspätung hatte.** → **Verspätung haben** 且動詞末位 **hatte**。  
9. **Ich erfuhr, dass sie krank ist.** **錯** → **正** **Ich erfuhr, dass sie krank war.** → **erfuhr** 過去框架下從句用 **war**。  
10. **Sie antwortete, dass sie wird später kommen.** **錯** → **正** **Sie antwortete, dass sie später kommen würde.** → 情態動詞框式，**würde** 與 **kommen** 分居尾位。  
11. **Ich denke, Dass das stimmt.** **錯** → **正** **Ich denke, dass das stimmt.** → **dass** 小寫。  
12. **Er findet, dass zu teuer es ist.** **錯** → **正** **Er findet, dass es zu teuer ist.** → 從句語序 **es** + 謂語末 **ist**。  
13. **Sie sagt, sie bleibt.**（表間接）**錯** → **正** **Sie sagt, dass sie bleibt.** → 教學階段加 **dass** 較穩。  
14. **Wir hoffen, dass aufhört es.** **錯** → **正** **Wir hoffen, dass es aufhört.** → 主語 **es** 在前，**aufhört** 尾位。  
15. **Sie befürchtet, es regnet.** **錯** → **正** **Sie befürchtet, dass es regnet.** → 加 **dass** 標明從句。  
16. **Er sagte, dass ich dir helfen würde.**（指他幫你）**錯** → **正** **Er sagte, dass er mir helfen würde.** → 轉述時人稱依敘述視角改 **er／mir**。  
17. **Sie sagten, dass wir da sind.** **錯** → **正** **Sie sagten, dass sie da seien/wären.** → 原話 **wir** 在轉述中改 **sie**；正式形見 **10-8**。  
18. **Ich sagte, dass du pünktlich warst.** **錯** → **正** **Ich sagte, dass er pünktlich war.** → 原 **du** 在旁述中改 **er**（依題幹指涉）。  
19. **Sie sagte, dass ich fertig war.** **錯** → **正** **Sie sagte, dass sie fertig war.** → 女說者 **Ich**→**sie**。  
20. **Er meinte, dass es regnet.**（主句過去、轉述當時在下雨）**錯** → **正** **Er meinte, dass es regnete.** → 退一步用 **regnete** 與 **meinte** 呼應。

## Practice

### 文法與詞彙選擇（四選一）

**Q1.** **Er sagt, ___ müde ist.**  
A. er　B. dass er　C. wenn er　D. ob er  
||**正解：B**||

**Q2.** **Sie meinte, ___ der Plan gut war.**  
A. dass　B. der　C. wenn　D. ob  
||**正解：A**||

**Q3.** **Wir hören, ___ der Zug Verspätung hat.**  
A. der Zug hat　B. dass der Zug hat　C. dass der Zug Verspätung hat　D. hat der Zug  
||**正解：C**||

**Q4.** **Er sagte, ___ keine Zeit ___.**  
A. er … hat　B. dass er … hatte　C. er … hatte　D. dass hat er  
||**正解：B**||

**Q5.** **Ich glaube, ___ stimmt.**  
A. das　B. dass das　C. wenn das　D. ob das  
||**正解：B**||

**Q6.** **Sie antwortete, ___ später kommen ___.**  
A. sie … wird　B. dass sie … würde　C. sie … würde　D. dass würde sie  
||**正解：B**||

**Q7.** **Er findet, ___ zu teuer ist.**  
A. es　B. dass es　C. wenn es　D. ob es  
||**正解：B**||

**Q8.** **Wir hoffen, ___ bald aufhört.**  
A. es regnet　B. dass es regnet　C. dass es mit dem Regen aufhört　D. aufhört es  
||**正解：C**||

**Q9.** **Sie erzählte, ___ Urlaub ___.**  
A. sie macht　B. dass sie machte　C. dass sie Urlaub machte　D. machte sie Urlaub  
||**正解：C**||

**Q10.** **Ich denke, ___ es möglich ist.**  
A. dass　B. weil　C. ob　D. wenn  
||**正解：A**||

**Q11.** **Sie befürchtet, ___ regnet.**  
A. es　B. dass es　C. wenn es　D. ob es  
||**正解：B**||

**Q12.** **Er erzählte, ___ krank ___.**  
A. sie ist　B. dass sie war　C. sie war　D. dass war sie  
||**正解：B**||

### 句構與語序（完整句四選一）

**Q13.**  
A. Er sagt, dass er müde ist.  
B. Er sagt, dass ist er müde.  
C. Er sagt, er dass müde ist.  
D. Er sagt dass, er müde ist.  
||**正解：A**||

A. Sie meinte, dass der Plan gut war.  
B. Sie meinte, dass war der Plan gut.  
C. Sie meinte der Plan, dass gut war.

||**正解：A**||

**Q15.**  
A. Wir hörten, dass der Zug Verspätung hatte.  
B. Wir hörten, dass hatte der Zug Verspätung.  
C. Wir hörten der Zug, dass Verspätung hatte.  
D. Wir hörten, der Zug dass Verspätung hatte.  
||**正解：A**||

||**正解：A**||

## Quick Review

- **dass**-從句：**連接詞起頭、動詞句末**。  
- 主句過去時，從句常**退一步**（依書表）。  
- **Konjunktiv I** 正式間接：**10-8**。
