# **werden** 現在式人稱變化

**Stage 2 · Unit 2-4 · Lektion 6** — 動詞 ***werden*** 現在式：**werde／wirst／wird／werden／werdet**（本課：變位＋「變成」初階；未來式標記後續再加深）

---

## Learning Goal

- 能默寫或口誦 ***werden*** 現在式六個人稱形式（對照 **§3.3 L6 必提**）。  
- 能讀懂 ***werden + Nom.*** 表「變得／成為」的初級句。  
- 知道 ***werden*** 在後續課會出現「未來式助動詞」「被動」等用法，本課先打好**現在變位**。

---

## Core Concept

### *werden* 為什麼重要？

它是高頻**不規則**動詞：現在式 **ich werde、du wirst、er wird** 與 **sein／haben** 一樣需要整組記。初級常見義項之一是「變成、成為」：**Er wird Arzt.** 日後會學 **Ich werde lernen.**（未來）與被動 **Es wird gemacht.**，變位**同一套**，現在記熟最划算。

### 本課邊界

- **只練現在式變位**＋**werden + 補語（名詞／形容詞）** 表變化。  
- **不定詞句框、被動**到 **Stage 4／8** 再系統講。

---

## Key Forms / Rules

### *werden* 現在式變位表

| 人稱 | 變位 |
|------|------|
| ich | **werde** |
| du | **wirst** |
| er / sie / es | **wird** |
| wir | **werden** |
| ihr | **werdet** |
| sie / Sie | **werden** |

### 常見結構（初級）

- **… wird / werden + 名詞（補語）**：*Sie wird Ärztin.*  
- **… wird / werden + 形容詞**：*Es wird kalt.*  
- **時間副詞**可放句首或動詞後（**2-5–2-7** 會再強化動詞位置）。

## Example Sentences

### ① 身分／職業「成為」

**Ich werde Arzt.** （我成為醫師／要當醫師。）  
**Du wirst Lehrerin.** （你成為女老師。）  
**Er wird Ingenieur.** （他成為工程師。）  
**Sie wird Studentin.** （她成為女學生。）  
**Wir werden Freunde.** （我們成為朋友。）

### ② 狀態「變得」

**Es wird kalt.** （天變冷了。）  
**Es wird warm.** （天變暖了。）  
**Du wirst müde.** （你變累了。）  
**Das Wetter wird schlecht.** （天氣變糟。）  
**Wir werden fertig.** （我們快好了／變成完成狀態。）

### ③ 變化過程（簡單敘述）

**Das Kind wird groß.** （小孩長大。）  
**Die Pflanze wird grün.** （植物變綠。）  
**Er wird stark.** （他變強壯。）  
**Sie wird ruhig.** （她變安靜。）  
**Ihr werdet schnell.** （你們變快了。）

### ④ 疑問與簡答

**Wirst du müde?** — **Ja, ich werde müde.** （你會不會累？— 會。）  
**Wird es warm?** — **Ja, es wird warm.** （會變暖嗎？— 會。）  
**Werdet ihr fertig?** — **Ja, wir werden fertig.** （你們會好嗎？— 會。）  
**Wird sie Ärztin?** — **Ja, sie wird Ärztin.** （她會當醫師嗎？— 對。）  
**Werden Sie Lehrer?** — **Ja, ich werde Lehrer.** （您會當老師嗎？— 對。）

## Quick Review

- *werden* 現在式：**ich werde, du wirst, er/sie/es wird, wir werden, ihr werdet, sie/Sie werden**。  
- **wird** 給 **er/sie/es**；**werdet** 給 **ihr**。  
- **2-5** 起專講**動詞第二位（支點）**與直述句。
