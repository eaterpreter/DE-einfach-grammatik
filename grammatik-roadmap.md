## 1. 整體目標

學習者能逐步達成：

1. 掌握德語基本 **語序**（動詞位置、框式結構入門）。
2. 使用 **現在式** 描述習慣與當下動作。
3. 理解 **性、數、格** 與冠詞、名詞、代詞搭配。
4. 運用常見 **介詞、連接詞、從句** 造句。
5. 辨認 **時態**（Präteritum、Perfekt 等）與 **被動、反身** 等基本概念。
6. 接觸 **Konjunktiv I/II** 與 **分詞、zu-Infinitiv** 等進階主題（依 Stage 逐步引入）。

## 2. 必教覆蓋（與書 Lektion 主題對齊＝核銷用）

以下與 **Lektion 1–40 標題層級**對齊，作為 **全課程必覆蓋** 核銷；細部講解粒度見 **§3.0**、逐條必提全文見 **§3.3**。

- **現在式與動詞**：規則／不規則變化、動詞位置、**sein / haben / werden**、助動詞與感官動詞、可分／不可分動詞。
- **疑問**：語序與疑問詞。
- **名詞與冠詞**：性別、定冠詞／不定冠詞變格、指示／否定／所有格冠詞、複數。
- **代詞**：人稱代名詞變格、反身、不定／指示代名詞、**es**、關係代名詞與關係副詞。
- **否定**：**nicht** 位置（入門與進階）。
- **句子連接**：對等連接詞、從屬連接詞、主從句語序。
- **形容詞**：變化、名詞化、比較級／最高級。
- **時態系統**：「時態」總覽、動詞三基本形式、**Präteritum**、**Perfekt**、**Plusquamperfekt / Futur II**（依課次）。
- **被動**、**反身**。
- **不定式**：**zu-Infinitiv** 與片語。
- **分詞**、**語式與助動詞**、**Konjunktiv II / I**、**數詞**。

## 3. 教學順序、粒度與書本核銷

### 3.0 Unit 拆分與例句密度（對齊 JP `basic-japanese-roadmap.md` §3）

- **規劃優先、書為參考**：書 **Lektion 1–40** 提供**標題與必提登錄**（§3.3）；**課表與產出順序**以本檔 **`stage-unit`** 與 **`docs/build-progress.md` §②** 為準。
- **非固定課表**：各 Stage 有 **建議 unit 數量區間**（§3.1）。可依認知負荷 **增列** unit；**不要**為維持「每 Stage 剛好四格」而硬把獨立主題併成一課。
- **一 unit 一可教微主題**：以「學完能自測、能做完一組對應練習」為準；若必提內含 **構成／語用／對比／圖示敘述** 等可分塊內容，**優先拆成多個 unit**。
- **例句密度（預設）**：每個 **Example Sentences** 之 **`###` 小節**至少 **5 句**德語例句，與 **`docs/lesson-template.md`** 一致；**Sentence Breakdown**、**Common Mistakes** 須同序、同數對齊。
- **介詞／格支配**：以「同一支配關係或同一義項群」為單位累積例句；每一義項群仍應達到**至少 5 句**可感知的德語例句（與上項小節下限一致；對齊 JP Stage 2「每個助詞至少 5 個初級例句」精神），再進入下一群或下一課。
- **暫不拆檔時**：單一 `.md` 內須用 **分層小節**，並遵守 **`docs/lesson-template.md`** 的 **小節最低例句數** 與 Breakdown／Mistakes 對齊規則。

**編號**：**`stage-unit`**（例如 `7-3`）。**`unit`** 為該 Stage 內 **遞增正整數**，可 **> 4**；檔名 **`{stage}-{unit}-{slug}.md`** 不變。

**跳關**：除非使用者明確要求，否則依 `docs/build-progress.md` §② **同 Stage 內由小到大**，且 **Stage N 仍有未產出列時不開 Stage N+1**。

### 3.2 重課拆段範本（產課時遵守；對應列見 build-progress）

#### 書 L27（過去完成式・未來完成式・完成式補強・時態圖）

以下通常 **各占 1 個 project unit**（Stage 7 內對應 **7-9–7-15**，見 `build-progress`）；必要時可再細拆，但 **不得** 省略核銷要點。

1. **Plusquamperfekt**：介紹與使用時機（「過去的過去」、與敘事主線之關係）。
2. **Plusquamperfekt**：構成＝人稱 + **完成式助動詞（haben/sein 的過去式）** + 過去分詞。
3. **Futur II**：介紹與使用時機。
4. **Futur II**：構成＝人稱 + *werden* + 過去分詞 + *haben/sein*（句末）。
5. **Perfekt**（與前接 L26 銜接）：補充定位／對照（若需獨立成課以避免單檔過長）。
6. **Perfekt**：例句與對照（與 Präteritum、敘事時間軸呼應）。
7. **德語時態示意**：時間軸敘述—**現在**為中樞；**向左**為過去軸（很久以前 → 過去某時間點 → 現在）：其中「現在 ↔ 過去某點」可見 **現在完成式／過去式**；「過去某點 ↔ 更遠的過去」為 **過去完成式**；**向右**為非過去（現在式 → 未來 → **未來完成式** 等）。**現在以左**可視為過去域、**以右**為非過去域（用語以講義圖示為準，與書圖一致核銷）。

  - 必提：**不規則動詞**（現在式等）。
- **L10** → 建議 **3-2–3-4**（§3.2 次優先）｜指示冠詞・否定冠詞・所有格冠詞  
  - 必提：**指示冠詞** *dieser / solcher / welcher / jeder* 等之各種變化。**否定冠詞 kein**（並與不定冠詞 *ein* 比較）之變化；**kein 與 nicht** 使用比較。**所有格冠詞** *mein / dein / ihr / sein / Ihr / unser / euer / ihr* 與各格變化。
- **L11** → 建議 **3-5–3-8**（§3.2 **書 L11**）｜介系詞 1  
  - 必提：書附**圖表**：支配二格或三格如 *trotz / wegen*；三格 *ab / außer / zu / bei / nach / von / aus / mit / seit / gegenüber*；可三／四格 *über / vor / auf / an / hinter / in / neben / unter / zwischen*；四格 *bis / durch / ohne / um / gegen / für*。單元內文另列：**二格** *während / wegen / trotz / statt*；**三格** *mit / aus / nach / von / zu / bei / seit / ab*；**四格** *für / durch / gegen / um / entlang / ohne / bis*。（與圖表並列時註明兩套整理之差異。）
- **L12** → 建議 **3-9–3-10**（§3.2 **書 L12**）｜介系詞 2  
  - 必提：可三／四格介詞在不同格之**意義比較**：*in / auf / unter / an / vor / hinter / über / neben / zwischen*。**介詞與定冠詞合併**：*im / ins / aufs / ans / an dem* 等。

#### Stage 4（書 L13–L16）

- **L13** → 建議 **4-1**｜人稱代名詞的變格  
  - 必提：人稱代詞外，含**二格用法**；**da + 介詞**：*darauf / daran / darin / davon* 等。
- **L14** → 建議 **4-2–4-6**（§3.2 **書 L14**）｜助動詞的用法與感官動詞  
  - 必提：*können / dürfen / mögen / müssen / sollen / wollen*；特殊 **möchte**；助動詞**否定句**；**單獨使用**助動詞；**未來式助動詞 werden**；**感官動詞**如 *sehen + 原形動詞* 等用法。
- **L15** → 建議 **4-7**｜原形動詞與動詞片語・nicht 的位置 1  
  - 必提：**動詞片語語序**；動詞片語與 **nicht**。
- **L16** → 建議 **4-8**｜不可分離動詞・可分離動詞 1  
  - 必提：**可分動詞**＝動詞本體＋前綴。**不可分動詞**＝詞綴＋動詞（如 *emp- / be- / ent- / er- / ge- / ver- / zer- / miss-* 等）；判準為**可分前綴本身可為獨立詞**。

#### Stage 5（書 L17–L20）

- **L17** → 建議 **5-1–5-4**（§3.2 **書 L17**）｜表示「請求・建議・命令」的命令式  
  - 必提：**du / ihr / Sie** 命令式；**sein / haben / werden** 命令式；**可分／不可分**命令式。語用**光譜**：最左疑問「？」→ 最右命令「！」；中間含請求／指示／建議（分水嶺）、獎勵／鼓勵、注意／警告、對無禮或瘋狂行為之反應、對寵物命令等。
- **L18** → 建議 **5-5**｜連接詞 1・對等連接詞  
  - 必提：**對等** *und / aber / oder / denn*。**從屬**如 *weil / obwohl / wenn* 等（依書列）。
- **L19** → 建議 **5-6**｜主句與從句・從屬連接詞 1  
  - 必提：主句／從句概念；複習**動詞位置**：**直述**句首－動詞－句尾；**是非疑問**動詞在句首；**W-疑問**疑問詞－動詞－句尾；**從句**連接詞起頭、動詞在句尾。例：*wissen*。
- **L20** → 建議 **5-7–5-12**（§3.2 **書 L20**）｜主句與從句・從屬連接詞 2  
  - 必提：以**例句**為主：**1a–1c** *Daher kommt… / Wegen der Krankheit… / Weil sie krank ist…*，焦點在 *kommt* 位置，比較 *daher / wegen… / weil…*。**2a–2c** *Trotzdem… / Trotz der Fristverlängerung… / Obwohl die Frist um ist…*；從句置前時主句以動詞開頭。**3a–3b** *wenn* 位置對調。**4a–4b** *ob* 從句與主句。*während / obwohl / da / wenn* 等補充例與用法。

#### Stage 6（書 L21–L24）

- **L21** → 建議 **6-1–6-3**（§3.2 次優先）｜形容詞的用法  
  - 必提：**形容詞變格**：**有定冠詞／無冠詞／有不定冠詞**三種狀況比較。
- **L22** → 建議 **6-4**｜形容詞的名詞化  
  - 必提：分**有定冠詞／有不定冠詞**；**中性形容詞名詞化**搭配 *etwas / nichts / viel* 等；被名詞化形容詞單字例；**陰／陽限人或生物（活物）**，**中性適用全部**。
- **L23** → 建議 **6-5**｜時態  
  - 必提：熟習**現在式、過去式、現在完成式**；並點出**未來式、過去完成式、未來完成式**存在但**口語頻率較低**。
- **L24** → 建議 **6-6**｜動詞的三種基本形式  
  - 必提：規則／不規則動詞之**過去式與過去分詞**；可分／不可分動詞之過去式與過去分詞（不可分**不**加 *ge-*、保留前綴）；以 **-ieren** 結尾動詞。

#### Stage 7（書 L25–L28）

- **L25** → 建議 **7-1–7-3**（§3.2 **書 L25**）｜過去式與用法  
  - 必提：過去式**人稱變化**、**句構**；助動詞過去式與人稱；過去式**使用情境**（媒體、新聞、雜誌、文學、童話等）。
- **L26** → 建議 **7-4–7-8**（§3.2 **書 L26**）｜現在完成式與用法  
  - 必提：與過去式**比較**（書列動詞如 *denken / fühlen / empfinden / meinen / sagen / rufen / schreien / singen / finden / hören / sehen / gehen / kommen / laufen / spielen / zeigen* 等）。**haben + PII**、**sein + PII**；動詞在現在完成式中於**直述／是非疑問／W-疑問／從句**的位置；可分／不可分與助動詞之現在完成式。
- **L27** → 建議 **7-9–7-15**（§3.2 **書 L27**；**勿**併成單一 unit 除非使用者明說速通）｜過去完成式・未來完成式・完成式補強・時態圖  
  - 必提：**過去完成式**介紹／使用時機；構成＝人稱＋完成式助動詞＋過去分詞。**未來完成式**介紹；構成＝人＋*werden*＋過去分詞＋*haben/sein*。**完成式**例句。書末**德語時態示意**（時間軸敘述見 §3.2）。
- **L28** → 建議 **7-16–7-20**（§3.2 **書 L28**）｜反身代名詞與反身動詞  
  - 必提：**sich**；反身代名詞與動詞表現（以 *legen / vorstellen* 為例）；**反身動詞**；**三格反身**；**相互代詞**；**sich lassen + 原形**＝請人代自己做。

#### Stage 8（書 L29–L32）

- **L29** → 建議 **8-1–8-5**（§3.2 **書 L29**）｜被動的語態  
  - 必提：主動／被動；被動**構成**（現在、過去、現在完成、過去完成、未來、帶情態動詞被動）；被動與動詞**位置**（直述、是非疑問＋過去式、W-疑問＋現在完成、從句＋現在完成、從句＋情態）；主被動對應（*von + 三格*）、**man** 作主詞、不及物動詞被動、**動作被動與狀態被動**。
- **L30** → 建議 **8-6–8-10**（§3.2 **書 L30**）｜es 的用法・疑問詞  
  - 必提：**es**：中性名詞指涉、指前文、非人稱主詞、天氣／時間／距離／生理心理等無法指定主詞之表達、其他非人稱。*es gibt + 四格*；*es handelt sich um*；*es dreht sich um*。**疑問詞** *was / wer / wo / woher / wohin / wann / warum / wie / was für ein / welcher*；*was* 與介詞縮合 *woran / worum / wovon* 等。
- **L31** → 建議 **8-11**｜不定代名詞・指示代名詞  
  - 必提：**不定代名詞**與**指示代名詞**介紹（依書展開）。
- **L32** → 建議 **8-12–8-13**（§3.2 **書 L32**）｜關係代名詞與用法  
  - 必提：**人稱／指示／關係代名詞**比較與差異；關係代名詞**各格例句**；造關係子句**步驟**；與**英語差異**。

#### Stage 9（書 L33–L36）

- **L33** → 建議 **9-1**｜關係副詞・不定關係代名詞  
  - 必提：例：*Das ist die Stadt, in der sie wohnt*／*… wo sie wohnt*。**不定關係** *was / wer*（分別表事物／人）。
- **L34** → 建議 **9-2–9-5**（§3.2 **書 L34**）｜比較級・最高級  
  - 必提：比較級於**謂語、名詞修飾、副詞**；最高級同上；*am jüngsten / der jüngste* 等；不規則如 *bald / gern* 等；同級 *so … wie …*；強調程度 *immer …*；*je + 比較級, desto + 比較級*（隨 1 加劇 2 跟著加劇）。
- **L35** → 建議 **9-6–9-8**（§3.2 **書 L35**）｜zu 不定式・zu 不定式片語  
  - 必提：原形與 **zu 不定式**；動詞片語與 zu 不定式片語；片語作**主詞、受詞、謂語、名詞修飾、帶介詞**；**不加逗號**情況：*brauchen + nicht + zu / scheinen + zu / haben + zu / sein + zu / pflegen + zu* 等。
- **L36** → 建議 **9-9–9-12**（§3.2 **書 L36**）｜分詞的世界  
  - 必提：**現在分詞**（名詞修飾、謂語、名詞、複合詞）；**過去分詞**（名詞修飾、謂語、名詞、副詞）；單獨使用過去分詞（指示、確認）；過去分詞於**完成式與被動**之差異。

#### Stage 10（書 L37–L40）

- **L37** → 建議 **10-1**｜語式・各語式的助動詞  
  - 必提：**語式**：直陳、命令（見 L17）、虛擬（L38–L39）；各語式**助動詞**用法。
- **L38** → 建議 **10-2–10-5**（§3.2 **書 L38**）｜假設語氣（虛擬式 II）  
  - 必提：假設說明與例（已發生用直陳、或然用虛擬）；直陳與 **Konjunktiv II** 比較；與過去事實相反；*würden + 原形*；*als ob + KII*；單用 KII（虛擬願望、省略前句、請求／建議／溫和請求）。
- **L39** → 建議 **10-6–10-9**（§3.2 **書 L39**）｜引用法（虛擬式 I）  
  - 必提：**虛擬式 I**：直接引語、間接引語（連接詞直述）、間接引語（虛擬引用）；如何構成 KI；直陳與 KI 比較；間接引用注意事項；祈禱語、注意事項等特殊情境。
- **L40** → 建議 **10-10**｜數詞  
  - 必提：**數字、序數、時間、日期**。

## 4. 必要輸出類型（單一 unit）

每課至少包含：

- 講義：目標、核心概念、例句、拆解、常見錯誤。
- 練習：選擇題、填空（選擇題化）、排序／語序判斷（四選一完整句）、必要時造句提示題。
- 複習：重點摘要、小測可選。

細部格式見 `docs/lesson-template.md`、`docs/exercise-rules.md`。單課內 **小節與例句下限** 見 `lesson-template.md`（與 **§3.0** 一致）。

**全專案一致品質（含已產出 unit 之回補）**：`units/` 內容須符合 **`docs/style-guide.md`**（**§3** 解說深度、格位 **德語名 + 縮寫**、台灣用語）與 **`docs/generation-brief.md` §3.1**；避免僅部分課次採高標、其餘維持舊口吻。

## 5. 內容設計原則

- 每課只引入有限新資訊；例句短、自然、生活化。
- 解說用 **繁體中文**；德語術語首次出現附白話說明。
- 避免尚未教過的文法大量堆在例句裡（除非標註為「先記句型」）。
